# Learning Path — Solar Energy Gathering & Storage

Use this as a checklist from **key hands-on** to **mid knowledge**.

## Phase A — Hands-on (integrated)

- [ ] Read `01_EcoFlow_Hands_On`
- [ ] Map every port on a real PPS (EcoFlow or peer)
- [ ] Full wall charge once; note Wh and behavior
- [ ] One solar charge session with Voc verified
- [ ] Run the same load on AC inverter vs USB/DC; compare drain
- [ ] Write your unit’s max solar Voc on a sticker

## Phase B — Identify

- [ ] Read `02_Identify_Components`
- [ ] With multimeter, measure a battery and a panel Voc
- [ ] Identify PWM vs MPPT on a photo or real unit
- [ ] Read one panel nameplate aloud (Pmax, Voc, Vmp, Isc)
- [ ] Name five connectors from `reference/connectors.md`

## Phase C — Safe assembly

- [ ] Read `03_Safe_Assembly`
- [ ] Recite connect order from memory: battery → controller → panel
- [ ] Size one fuse for a practice wire gauge
- [ ] Perform a dry-run assembly with covers on panels

## Phase D — Mixed brands

- [ ] Read `04_Mixed_Brand_Systems`
- [ ] Fill the mixed-brand worksheet for a hypothetical thrift panel + your PPS
- [ ] Explain why series Voc is the #1 silent killer
- [ ] List three mixes that are OK and three that are not

## Phase E — Market literacy

- [ ] Read `05_US_Brands_How_They_Work`
- [ ] Compare two retail PPS on Wh, Voc window, LFP, weight
- [ ] Explain PPS vs DIY architecture to someone else
- [ ] Name one premium DIY controller brand and one kit brand

## Phase F — Scrap build

- [ ] Read `06_Scrap_Electronics_Build`
- [ ] Gather or shop a minimum parts kit (even if new cheap PWM)
- [ ] Build 12 V panel → controller → SLA → LED
- [ ] Log voltages for a full day
- [ ] Optional: add USB buck; optional later: small inverter

## Phase G — Mid knowledge capstone

- [ ] Design on paper a mixed system: used panel + MPPT + LFP + inverter
- [ ] Complete `reference/load_planning_worksheet.md` for a 72-hour outage
- [ ] Create a “go bag power” list: PPS + one scrap spare path
- [ ] Teach connect order and polarity check to another person

## Done when you can answer without notes

1. What connects first on a DIY controller — battery or panel?  
2. Why does a “12 V panel” read ~20 V open circuit?  
3. Can you use a no-name panel on EcoFlow? What must match?  
4. Why not parallel lead-acid with LiFePO4?  
5. What is the energy path sun → AC lamp?  
6. What scrap battery chemistry do we start with and why?
