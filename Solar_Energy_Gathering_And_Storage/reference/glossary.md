# Glossary — Solar Energy Gathering & Storage

| Term | Definition |
|------|------------|
| **Ah** | Ampere-hours — battery capacity unit |
| **AGM** | Absorbent Glass Mat — sealed lead-acid type |
| **BMS** | Battery Management System — protects lithium packs |
| **Bulk / Absorb / Float** | Lead-acid charge stages |
| **C-rate** | Charge/discharge current relative to capacity (1C ≈ 100 A on 100 Ah) |
| **CC/CV** | Constant current / constant voltage charging |
| **Deep cycle** | Battery designed for repeated discharge (vs starter) |
| **DoD** | Depth of Discharge |
| **Isc** | Short-circuit current of a panel |
| **Imp** | Current at maximum power |
| **Inverter** | DC to AC converter |
| **LFP / LiFePO4** | Lithium iron phosphate chemistry |
| **Load** | Anything consuming power |
| **MC4** | Standard PV connector family |
| **MPPT** | Maximum Power Point Tracking charge controller |
| **NMC** | Nickel manganese cobalt lithium chemistry (some older PPS) |
| **OCV / Voc** | Open-circuit voltage |
| **Off-grid** | Not tied to utility grid |
| **Parallel** | + to +, − to − — capacity/current adds |
| **PPS** | Portable Power Station |
| **PWM** | Pulse Width Modulation controller (simpler) |
| **PV** | Photovoltaic (solar electric) |
| **SCC** | Solar Charge Controller |
| **Series** | + of one to − of next — voltage adds |
| **SLA** | Sealed Lead-Acid |
| **SoC** | State of Charge (%) |
| **STC** | Standard Test Conditions for panel ratings |
| **Vmp** | Voltage at maximum power |
| **Wh** | Watt-hours — energy (V × Ah ≈ Wh for rough DC math) |
| **XT60** | High-current DC connector common on PPS solar inputs |
