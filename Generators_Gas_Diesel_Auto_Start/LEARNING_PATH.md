# Learning Path — Generators Gas / Diesel / Auto-Start

## Phase A — Fuel literacy

- [ ] Read `01_Fuel_Types_Gas_Diesel`

- [ ] Pick the right fuel for: (1) city NG home (2) rural propane (3) farm diesel

## Phase B — Machine classes

- [ ] Read `02_Portable_vs_Standby`
- [ ] List your critical loads with running + surge watts
- [ ] Decide portable vs standby for your primary site

## Phase C — Auto-start

- [ ] Read `03_Auto_Start_ATS_Outage`
- [ ] Draw ATS sequence from memory (fail → start → transfer → return)
- [ ] Explain interlock vs ATS
- [ ] Schedule a start-battery check before winter

## Phase D — Canadian fit

- [ ] Read `04_Canadian_Brands_Best_Fit`
- [ ] Call or web-search **two** local installers (service radius)
- [ ] Verify CSA and permit path for your province
- [ ] Choose brand direction with installer SLA in mind

## Phase E — Cold & safety

- [ ] Read `05_Cold_Climate_Safe_Ops`

## Phase F — Hybrid

- [ ] Read `06_Hybrid_With_Solar`
