# Generators Cheat Sheet

## Connect safety

```
Portable → outdoor only → cords or inlet + interlock/transfer
Standby → permanent → ATS + licensed install
NEVER suicide cord into dryer outlet
```

## Auto-start sequence

```
Grid fail → delay → crank → stabilize V/Hz → ATS to gen
Grid OK → delay → ATS to utility → cool-down → stop
```

## Fuel pick (Canada)

| Site | First choice |
|------|----------------|
| Home on natural gas, unattended | NG standby + ATS |
| Rural, tanks already | Propane standby or dual-fuel portable |
| Farm / long continuous | Diesel |
| Cottage occasional | Honda-class inverter or dual-fuel |
| Condo | Battery / solar station (gen often banned) |

## Brand direction (not gospel)

| Need | Look at |
|------|---------|
| Standby + many techs | Generac, also quote Kohler |
| Premium standby | Kohler, Cummins |
| Best portable reliability | Honda, Yamaha |
| Dual-fuel value portable | Westinghouse, Generac, Champion (verify CSA) |

## Winter

- Start battery health  
- Fresh fuel / correct diesel blend  
- Clear snow  
- CO alarms  
- LP tank level  

## Hybrid fuel save

```
Battery/solar first → gen to recharge or for surge → shut gen
```
