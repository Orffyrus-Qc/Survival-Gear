# Glossary — Generators & ATS

| Term | Meaning |
|------|---------|
| **Alternator** | Makes AC from engine rotation |
| **AMF** | Automatic Mains Failure controller |
| **ATS** | Automatic Transfer Switch |
| **Bound neutral** | Neutral bonded to ground at source — must match transfer design |
| **CSA** | Canadian Standards Association listing mark (common approval path) |
| **CEC** | Canadian Electrical Code |
| **Crank** | Starter turns engine to start |
| **Derate** | Reduced power on LP/NG/altitude/temp |
| **Dual-fuel** | Gasoline + propane capable |
| **Exercise** | Scheduled test run of standby |
| **Inverter generator** | Electronics produce cleaner, often quieter AC |
| **Interlock** | Mechanical device preventing utility + gen breakers together |
| **kW / kVA** | Power ratings (pf matters for kVA) |
| **LP** | Liquid propane |
| **NG** | Natural gas |
| **Open transition** | Break utility before make generator (common residential) |
| **RBQ** | Régie du bâtiment du Québec (licensing context) |
| **Surge / starting watts** | Short motor-start capacity |
| **THD** | Total harmonic distortion — power cleanliness |
| **Transfer switch** | Switches load between utility and generator |
| **Wet stacking** | Diesel carboning from chronic light load |
