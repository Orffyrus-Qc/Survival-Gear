# 04 — Marques et meilleur choix pour le Canada

Il existe peu de génératrices grand public purement « made in Canada ». Le **meilleur choix pour le Canada**, c’est :

1. Marquage **CSA** (ou équivalent) et chemin d’installation légal  
2. **Concessionnaire / pièces / techs** près de chez vous (critique en hiver)  
3. Carburant aligné sur la logistique **GN / propane / diesel** locale  
4. Comportement **grand froid** et design d’enceinte  
5. Support **français** (manuels/UI) au besoin (Québec)  

---

## 1. Comment choisir (ordre de priorité)

```
1. Rôle portable vs secours
2. Carburant dispo sur le site (GN / LP / essence / diesel)
3. kW pour les charges critiques
4. Démarrage auto + ATS requis ? (maison sans surveillance)
5. Installateur certifié local + garantie
6. Réputation de marque / bruit / fonctions
```

La marque vient **après** le carburant et le rôle — même philosophie que le multi-marques solaire.

---

## 2. Secours résidentiel — marques courantes au Canada

| Marque | Position marché | Meilleur quand… | Notes Canada |
|--------|-----------------|-----------------|--------------|
| **Generac** | Volume résidentiel le plus installé en Amérique du Nord | Large réseau techs, bon rapport fonctions/prix | Souvent le plus facile à faire réparer ; ligne Guardian GN/LP ; Wi‑Fi courant |
| **Kohler** (Kohler Energy / Rehlko) | Premium résidentiel / léger commercial | Ingénierie premium, ownership long terme | Forte réputation ; confirmer techs certifiés locaux avant d’acheter |
| **Cummins** | Premium / ADN commercial | Fiabilité, sites exigeants | Pedigree industriel ; réseau parfois plus commercial |
| **Briggs & Stratton** | Concurrent résidentiel | Comparer les soumissions vs Generac/Kohler | Dispo variable par province |
| **Caterpillar / industriel** | Diesel commercial | Ferme, usine, télécom | Via dealers industriels (ex. Cat régionaux) — pas le kit banlieue typique |

### Résumé secours (Canada)

| Situation | Direction forte par défaut |
|-----------|----------------------------|
| Maison banlieue **avec gaz naturel**, veut auto-panne | **Generac ou Kohler** classe 14–22 kW + ATS — choisir par **qualité d’installateur local** et soumission, pas les guerres Internet |
| Rural **propane** seulement | Mêmes marques en **LP** ; dimensionner la cuve multi-jours + appareils LP |
| Éloigné / logistique carburant long / ferme | Envisager **diesel** industriel + ATS correct (Cummins/Cat…) |
| Marketing « 22 kW toute la maison » | Vérifier le design **sous-panneau critique** ; le chauffage électrique fait échouer le rêve |

**Installateur > badge :** un Generac bien mis en service bat un Kohler négligé (et l’inverse). Demander : permis, plan batterie de démarrage hiver, calcul de charge, délai de service en tempête.

## 3. Portables — marques courantes au Canada

| Marque | Force | Meilleur pour |
|--------|-------|---------------|
| **Honda** | Fiabilité, revente, culture de watts réels | Portable principal, chalet, électronique ; classe EU = référence |
| **Yamaha** | Onduleurs silencieux de qualité | Même palier que Honda |
| **Westinghouse** | Dual-fuel onduleur bon rapport | Kit tempête ROI grande surface / en ligne |
| **Generac** | Familiarité de marque, dual-fuel | Si vous aimez déjà l’écosystème |
| **Champion** | Watts/$ courant grande surface | Budget ; entretenir soigneusement |
| **Firman / DuroMax / Pulsar** etc. | Dual/tri-fuel valeur | OK si QC variable acceptée ; vérifier marques CSA |
| **Predator (Harbor Freight)** | Watts pas chers | Moins idéal comme seul secours médical/maison |

### Résumé portable (Canada)

| Situation | Direction |
|-----------|-----------|
| Électronique silencieuse + rendement | **Honda** ou **Yamaha** onduleur |
| Kit tempête dual-fuel budget | **Westinghouse / Generac / Champion** dual-fuel (CSA, vraies crêtes) |
| Chantier / outils | Open-frame ; diesel portable si vous roulez déjà diesel |
| Charger EcoFlow / stations | Génératrice à onduleur préférée |

## 4. Réalité « marque canadienne »

| Ce qu’on veut | Ce qui existe |
|---------------|---------------|
| Marque génératrice à feuille d’érable dominante | Rare en grand public |
| **Détaillants** canadiens | Canadian Tire, Home Depot Canada, Rona/Réno-Dépôt, Costco CA, dealers spécialisés |
| Écosystème **propane / gaz** canadien | Codes provinciaux + distributeurs locaux — souvent le vrai partenaire « local » |
| **Entrepreneurs électriciens** canadiens | Le partenaire critique pour ATS/secours |
| Industriel | Wajax, Finning (Cat), dealers Cummins, etc. |

**Meilleur fit canadien = marque nord-américaine + installateur certifié canadien + équipement CSA + contrat carburant fiable en janvier.**

---

## 5. Notes spécifiques Québec

- Pannes d’hiver + **verglas** : prioriser **secours démarrage auto** ou plan portable béton **avant** la tempête.  
- **Propane** courant hors réseaux GN.  
- Confirmer marquages **CSA** ; les imports gris peuvent échouer l’inspection.  
- Embaucher **licenciés** électricien / monteur gaz ; catégories **RBQ** pour beaucoup de travaux.  
- Froid : chauffe-batterie / bonnes batteries de démarrage, déneigement autour de l’enceinte, dimensionnement détendeur LP.  
- Documentation française : demander manuels FR et langue d’appli.  
- Pannes Hydro-Québec : le GN reste souvent disponible → secours GN brille ; toujours détecteurs CO et plan B si pression gaz chute.

## 6. Matrice marque par cas d’usage

| Cas d’usage | Carburant | Démarrage auto | Direction marque |
|-------------|-----------|----------------|------------------|
| Maison ville, GN, médical/frigo/fournaise | GN | **Oui ATS** | Generac / Kohler / Cummins via top installateur local |
| Maison rurale, chauffage propane | LP | **Oui ATS** | Idem ; coordonner cuve avec fournisseur propane |
| Chalet week-ends | Essence ou dual-fuel | Manuel OK | Honda EU ou dual-fuel onduleur moyen |
| Ferme multi-jours | Diesel | ATS ou manuel | Diesel industriel + transfert correct |
| Appartement / condo | — | Station batterie | Classe EcoFlow (module solaire) ; gén souvent interdite |
| Hybride solaire + gen | Essence/LP onduleur | Manuel ou smart start | Honda/qualité onduleur + discipline de transfert |

## 7. Liste d’achat (Canada)

- [ ] CSA / approuvé marché canadien  
- [ ] kW continus sur **votre** carburant (derate LP !)  
- [ ] 120/240 V si pompes 240 V / sécheuses  
- [ ] Pièces + tech locaux (« Qui service ça dans mon coin ? »)  
- [ ] Garantie **au Canada** (qui l’honore)  
- [ ] Monitoring appli dans votre langue  
- [ ] Options enceinte / paquet froid  
- [ ] Assez silencieux pour les limites de lot  
- [ ] Calcul de charge écrit pour secours  
- [ ] Chemin de permis clair

## 8. Generac vs Kohler vs Cummins (pratique, pas religion)

| Facteur | Generac | Kohler | Cummins |
|---------|---------|--------|---------|
| Densité dealers (plusieurs régions) | Souvent la plus haute | Bonne | Bonne, plus commercial parfois |
| Position prix | Valeur–milieu | Premium | Premium |
| Fonctions / Wi‑Fi | Très courant | Fort | Fort |
| Ce qui compte vraiment | Compétence installateur + entretien | Idem | Idem |

**Style de reco de ce projet :** obtenir **deux soumissions** (ex. Generac et Kohler) d’installateurs qui font calcul de charge + mise en service climat froid. Choisir le forfait avec le meilleur SLA de service, pas la brochure glossy.
