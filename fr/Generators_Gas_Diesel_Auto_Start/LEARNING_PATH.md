# Parcours d’apprentissage — Génératrices essence / diesel / démarrage auto

## Phase A — Littératie carburant

- [ ] Lire `01_Fuel_Types_Gas_Diesel`
- [ ] Expliquer essence vs diesel vs propane vs GN en une minute chacun
- [ ] Choisir le bon carburant pour : (1) maison ville GN (2) rural propane (3) ferme diesel

## Phase B — Classes de machines

- [ ] Lire `02_Portable_vs_Standby`
- [ ] Lister vos charges critiques avec W continus + crête
- [ ] Décider portable vs secours pour votre site principal

## Phase C — Démarrage auto

- [ ] Lire `03_Auto_Start_ATS_Outage`
- [ ] Dessiner la séquence ATS de mémoire (panne → start → bascule → retour)
- [ ] Expliquer interverrouillage vs ATS
- [ ] Planifier un check batterie de démarrage avant l’hiver

## Phase D — Fit canadien

- [ ] Lire `04_Canadian_Brands_Best_Fit`
- [ ] Appeler ou chercher **deux** installateurs locaux (rayon de service)
- [ ] Vérifier CSA et chemin de permis pour votre province
- [ ] Choisir une direction de marque avec le SLA d’installateur en tête

## Phase E — Froid et sécurité

- [ ] Lire `05_Cold_Climate_Safe_Ops`
- [ ] Test détecteur CO
- [ ] Audit stockage carburant
- [ ] Plan de déneigement pour l’emplacement gen

## Phase F — Hybride

- [ ] Lire `06_Hybrid_With_Solar`
- [ ] Écrire votre plan en couches : batterie / solaire / gen
- [ ] Pratiquer une charge de station par génératrice (setup extérieur sûr)
