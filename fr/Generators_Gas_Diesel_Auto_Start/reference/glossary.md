# Glossaire — Génératrices et ATS

| Terme | Sens |
|-------|------|
| **Alternateur** | Produit le CA à partir de la rotation moteur |
| **AMF** | Automatic Mains Failure — contrôleur de panne réseau |
| **ATS** | Automatic Transfer Switch — commutateur de transfert automatique |
| **Neutre lié (bound/bonded neutral)** | Neutre lié à la terre à la source — doit matcher le design de transfert |
| **CSA** | Canadian Standards Association — marquage d’approbation courant |
| **CEC** | Code canadien de l’électricité |
| **Crank** | Le démarreur tourne le moteur pour démarrer |
| **Derate** | Puissance réduite sur LP/GN/altitude/température |
| **Dual-fuel** | Capable essence + propane |
| **Exercice** | Course de test planifiée du secours |
| **Génératrice à onduleur (inverter)** | Électronique produit un CA plus propre, souvent plus silencieux |
| **Interverrouillage (interlock)** | Dispositif mécanique empêchant utilité + gen ensemble |
| **kW / kVA** | Notes de puissance (le facteur de puissance compte pour kVA) |
| **LP** | Propane liquide |
| **GN / NG** | Gaz naturel |
| **Transition ouverte (open transition)** | Couper utilité avant de faire génératrice (courant résidentiel) |
| **RBQ** | Régie du bâtiment du Québec (contexte de licences) |
| **Watts de crête / démarrage** | Capacité courte de démarrage moteur |
| **THD** | Distorsion harmonique totale — propreté de l’alimentation |
| **Commutateur de transfert** | Bascule la charge entre utilité et génératrice |
| **Wet stacking** | Carbonisation diesel par charge chronique trop faible |
