# Aide-mémoire génératrices

## Sécurité de raccordement

```
Portable → dehors seulement → cordons ou entrée + interverrouillage/transfert
Secours → permanent → ATS + installation licenciée
JAMAIS de cordon suicide dans la prise de sécheuse
```

## Séquence démarrage auto

```
Panne réseau → délai → crank → stabiliser V/Hz → ATS vers gen
Réseau OK → délai → ATS vers utilité → refroidissement → arrêt
```

## Choix carburant (Canada)

| Site | Premier choix |
|------|----------------|
| Maison sur gaz naturel, sans surveillance | Secours GN + ATS |
| Rural, cuves déjà là | Secours propane ou portable dual-fuel |
| Ferme / long continu | Diesel |
| Chalet occasionnel | Onduleur classe Honda ou dual-fuel |
| Condo | Station batterie / solaire (gen souvent interdite) |

## Direction marques (pas parole d’évangile)

| Besoin | Regarder |
|--------|----------|
| Secours + beaucoup de techs | Generac, aussi soumissionner Kohler |
| Secours premium | Kohler, Cummins |
| Meilleure fiabilité portable | Honda, Yamaha |
| Dual-fuel portable valeur | Westinghouse, Generac, Champion (vérifier CSA) |

## Hiver

- Santé batterie de démarrage  
- Carburant frais / bon mélange diesel  
- Déneiger  
- Détecteurs CO  
- Niveau cuve LP  

## Économie hybride

```
Batterie/solaire d’abord → gen pour recharger ou pour la crête → couper la gen
```
