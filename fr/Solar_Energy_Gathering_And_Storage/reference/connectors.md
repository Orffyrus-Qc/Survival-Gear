# Référence rapide des connecteurs

| Connecteur | Usage typique | Notes |
|------------|---------------|-------|
| **MC4** | Strings de panneaux | Étanche si bien assis ; outil de déverrouillage |
| **Dérivation MC4 (Y)** | Parallèle de panneaux | Fusibles de string si plusieurs en parallèle |
| **XT60** | Solaire/auto classe EcoFlow, hobby fort courant | Clé de polarité ; vérifier les adaptateurs |
| **XT90** | Plus de courant que XT60 | Mêmes habitudes |
| **Anderson SB50/SB175** | Treuil, DIY CC, certaines stations | Boîtier sans genre ; contacts ont un genre |
| **Anderson Powerpole** | Radioamateur, CC modulaire | Standard couleur rouge = + |
| **SAE 2 broches** | Mainteneurs, certains kits solaires | Facile d’inverser — multimètre |
| **Barrel DC5521** | Ports accessoires sur certaines PPS | Vérifier polarité du centre |
| **Prise allume-cigare** | Accessoires 12 V | Mauvais pour fort courant continu |
| **Cosse anneau** | Plots batterie, onduleurs | Qualité de sertissage = tout |
| **IEC C13/C14** | Matériel CA | Pas pour le CC batterie brut |

## Règle des adaptateurs

Chaque adaptateur est une chance d’inverser la polarité.  
**Continuité + tension au multimètre avant le premier branchement sur du matériel cher.**
