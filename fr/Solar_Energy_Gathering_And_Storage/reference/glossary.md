# Glossaire — Énergie solaire collecte et stockage

| Terme | Définition |
|-------|------------|
| **Ah** | Ampères-heures — capacité batterie |
| **AGM** | Absorbent Glass Mat — plomb étanche |
| **BMS** | Battery Management System — protège les packs lithium |
| **Bulk / Absorb / Float** | Étapes de charge plomb |
| **C-rate** | Courant relatif à la capacité (1C ≈ 100 A sur 100 Ah) |
| **CC/CV** | Courant constant / tension constante |
| **Cycle profond** | Batterie conçue pour décharges répétées (vs démarreur) |
| **DoD** | Profondeur de décharge |
| **Isc** | Courant de court-circuit d’un panneau |
| **Imp** | Courant à puissance maximale |
| **Onduleur (inverter)** | Convertisseur CC vers CA |
| **LFP / LiFePO4** | Chimie lithium fer phosphate |
| **Charge (load)** | Tout ce qui consomme de l’énergie |
| **MC4** | Famille de connecteurs PV standard |
| **MPPT** | Maximum Power Point Tracking — contrôleur de charge avancé |
| **NMC** | Chimie nickel manganèse cobalt (certaines anciennes PPS) |
| **OCV / Voc** | Tension en circuit ouvert |
| **Hors réseau (off-grid)** | Non relié au réseau électrique |
| **Parallèle** | + avec +, − avec − — capacité/courant s’ajoutent |
| **PPS** | Portable Power Station — station d’alimentation portable |
| **PWM** | Pulse Width Modulation — contrôleur plus simple |
| **PV** | Photovoltaïque |
| **SCC** | Solar Charge Controller — contrôleur de charge solaire |
| **Série** | + de l’un au − du suivant — tension s’ajoute |
| **SLA** | Sealed Lead-Acid — plomb étanche |
| **SoC** | State of Charge — état de charge (%) |
| **STC** | Conditions de test standard des panneaux |
| **Vmp** | Tension à puissance maximale |
| **Wh** | Watts-heures — énergie (V × Ah ≈ Wh en CC approximatif) |
| **XT60** | Connecteur CC fort courant courant sur entrées solaires PPS |
